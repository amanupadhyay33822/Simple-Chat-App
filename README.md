# Simple Chat App

A real-time chat application built using React, Socket.IO, and Node.js.

## Features

- Real-time communication using Socket.IO
- Simple and intuitive user interface


## Technologies Used

- **Front End:** React
- **Back End:** Node.js with Socket.IO
- **Styling:** MATERIAL UI
